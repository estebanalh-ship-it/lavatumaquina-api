## 📋 Descripción

**Lava Tu Máquina** es un sistema web de gestión integral desarrollado específicamente para el lavado de autos ubicado en Rengo, Chile. La plataforma permite a los clientes agendar servicios de lavado especializados y acceder a servicios mecánicos, mientras proporciona herramientas administrativas completas para la gestión del negocio.

### ✨ Características Principales

✅ **Sistema de Agendamiento Inteligente**
- Reservas en línea 24/7
- Selección de tipos de lavado (general y premium)
- Gestión de horarios y disponibilidad

✅ **Panel Administrativo Avanzado**
- Gestión centralizada para dueños y socios
- Control de citas y clientes
- Reportes y estadísticas
- Modulo cotizaciones.
- Modulo cambio de precios.

✅ **Interfaz Optimizada**
- Diseño responsive (móvil y escritorio)
- Experiencia de usuario intuitiva
- Navegación simplificada

## 🎯 Objetivo del Proyecto

Digitalizar completamente las operaciones del servicio automotriz "Lava Tu Máquina" para:
- **Automatizar** el proceso de reservas
- **Mejorar** la experiencia del cliente
- **Optimizar** la gestión administrativa
- **Escalar** el negocio de manera eficiente
- **Reducir** errores manuales en agendamiento
